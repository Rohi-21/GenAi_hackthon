Data scientists spend 60-80% of their time cleaning raw datasets. Build an agentic GenAI system that autonomously ingests a raw CSV dataset, profiles data quality issues (missing values, outliers, type mismatches, duplicates), applies cleaning actions with justifications, and generates a comprehensive data quality report with before-and-after statistics. Expected Deliverables: • CSV dataset upload and automated profiling • Issue detection across 6 quality dimensions • Autonomous cleaning with per-action justification log • Before-and-after data quality metrics comparison • Data quality report (DOCX) with methodology section Suggested Tech Stack: Python, pandas, LangChain Agents, LLM API, ydata-profiling, Streamlit Datasets Available: • Titanic Survival Dataset — 891 records with diverse data quality issues for cleaning demonstrations (kaggle.com/competitions/titanic) • UCI Adult Income Dataset — 48,842 records with missing values and categorical inconsistencies (archive.ics.uci.edu/dataset/2/adult) • Dirty Data Challenge Dataset (REIN) — Benchmark dataset for data cleaning algorithms with injected errors (github.com/BigDaMa/rein) • OpenRefine Test Datasets — Real-world messy datasets used in data cleaning workshops (openrefine.org/download)
